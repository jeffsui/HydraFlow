# 水滴助手(HydraFlow) - 项目概述

Drink Smart, Live Well

## 项目简介
一个现代化的Web端喝水提醒应用，帮助用户建立健康的饮水习惯。应用具有智能提醒、进度追踪、数据分析和个性化设置等功能。

## 页面功能规划

### 1. index.html - 主追踪界面
**布局结构:**
- 顶部导航栏: Logo + 页面切换 + 主题切换
- 简洁标题区: 应用名称 + 今日目标概览
- 主要内容区:
  - 左侧(40%): 水位动画显示 + 今日统计
  - 右侧(60%): 快速记录按钮 + 提醒设置
- 底部: 成就展示 + 激励信息

**核心功能:**
- 实时水位动画显示当前进度
- 一键快速记录喝水量
- 智能提醒系统控制
- 今日目标完成度展示

### 2. stats.html - 数据分析页面
**布局结构:**
- 导航栏: 与主页保持一致
- 统计概览: 关键指标卡片
- 图表区域:
  - 今日24小时喝水分布图
  - 近7天趋势图
  - 月度统计热力图
- 历史记录: 可筛选的数据列表

**核心功能:**
- 多维度数据可视化
- 历史记录查看和管理
- 喝水习惯分析
- 数据导出功能

### 3. settings.html - 个性化设置页面
**布局结构:**
- 导航栏: 与其他页面保持一致
- 设置分类:
  - 个人基本信息
  - 提醒设置
  - 目标设置
  - 界面主题
  - 音效设置
- 预览区域: 实时效果预览

**核心功能:**
- 个性化配置管理
- 主题切换预览
- 提醒音效测试
- 数据备份与恢复

## 技术实现方案

### 前端技术栈
- **基础**: HTML5 + CSS3 (Tailwind CSS)
- **动画**: Anime.js + p5.js + Matter.js
- **图表**: ECharts.js
- **特效**: Shader-park + PIXI.js
- **轮播**: Splide.js
- **字体**: Noto Sans SC + Noto Serif SC

### 数据存储
- **本地存储**: localStorage 存储用户设置和历史数据
- **数据结构**: JSON格式，支持导入导出
- **备份机制**: 自动备份重要数据

### 核心算法
- **提醒算法**: 基于用户习惯的智能提醒间隔
- **目标计算**: 根据体重、活动量计算推荐饮水量
- **数据分析**: 趋势分析、完成率统计

## 用户体验设计

### 交互流程
1. **首次使用**: 引导用户设置基本信息
2. **日常使用**: 一键记录 + 自动提醒
3. **数据查看**: 多维度统计图表
4. **个性化**: 丰富的自定义选项

### 响应式设计
- **桌面端**: 1920x1080 最佳体验
- **平板端**: 768px 以上适配
- **移动端**: 375px 以上支持

### 性能优化
- **懒加载**: 图表和动画按需加载
- **缓存策略**: 静态资源缓存
- **数据优化**: 压缩存储历史数据
